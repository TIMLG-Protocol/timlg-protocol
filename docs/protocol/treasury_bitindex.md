# Treasury & BitIndex

This section describes the treasury model and the BitIndex concept at a spec level.

## Treasury principles

- Transparent rules for how funds are used
- Predictable budgeting for:
  - infrastructure
  - audits / security reviews
  - grants / ecosystem incentives (if any)

## BitIndex (concept)

BitIndex is a structured mechanism to:
- track economic signals,
- apply allocation rules,
- support long-term sustainability.

!!! note
    Implementation details (signers, operational addresses, automation scripts) remain private until security hardening is complete.
